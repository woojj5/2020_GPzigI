## 구성파일 설명

app.py : flask 웹 구현

saves.ipynb : 웹 크롤링, 잔여학점 계산 작업 후 결과값을 리턴

counter.py : 접속자 수 카운터

## 나의 역할
- 기본적인 프로젝트 구조 기획
- selenium을 이용한 크롤링 구현 보조
- 웹 구현
- 클라우드를 이용한 프로젝트 배포

## 사용한 tools & library

웹 크롤링 : Selenium

잔여학점 계산 : Python

웹사이트 구현 : Flask

배포 : Nginx, AWS(EC2, ubuntu)
